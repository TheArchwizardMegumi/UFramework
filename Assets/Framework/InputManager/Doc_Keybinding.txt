[1]Default key binding:
	Edit Customization\KeySet.cs to set default key binding, adding new keys in enum KeyName and initialize them in KeySet.InitializeKeySet()

[2]Remember to use KeyBinding.LoadKeyBindingFromPlayerPref at the start of the game

[3]Set / Change a key:
	Use KeyBinding.SetKeyBinding. e.g: KeyBinding.SetKeyBinding(KeyName.interact, KeyCode.F) //Change interact operation from key "E" by default to "F"

[4]Get key state:
	Use InputManager.GetKey, GetKeyUp, GetKeyDown to get key code by operation name. Replace Input.GetKey and else.

[5]Save all change:
	UseKeyBinding.SaveKeyBinding.
